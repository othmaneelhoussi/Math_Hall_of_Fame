# Math Hall of Fame
This project is a simple Flask web application that displays profiles of famous mathematicians and their contributions. Users can also edit and update the information directly from the app.

Features:
- View Mathematicians – Browse profiles with names, birth years, bios, and contributions.
- Images & Links – Each mathematician has an image and links to similar figures.
- Edit Profiles – Modify details through an /edit/<id> page with pre-filled input fields.

Built with Flask, Jinja2, JQuery and Bootstrap for a clean and responsive design.
